## Ghaim (غَيْم)

Ghaim is a sensible and simple bilingual blogging template.

## Features

1. Bilingual
2. Simple CSS
3. KaTeX

## Installation

```bash
git clone https://github.com/mstfelg/ghaim
cd ghaim
npm run build
```

Or to start a browsersync session:

```bash
npm run serve
```

and

```
npm run serve:ar
```

for Arabic.
